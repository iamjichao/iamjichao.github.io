### @font-face

> CSS at-rule [@font-face](https://developer.mozilla.org/zh-CN/docs/Web/CSS/@font-face) 指定一个用于显示文本的自定义字体，字体能从远程服务器或者用户本地安装的字体中加载。如果提供了 local() 函数，从用户本地查找指定的字体名称，并且找到了一个匹配项，本地字体就会被使用，否则字体就会使用 url() 函数下载的资源。
